# Changelog

All notable changes to the Home Assistant Development Power will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.1] - 2026-03-08

### Added
- 🏠 icon for better visual presentation in Kiro Powers UI
- CHANGELOG.md following Keep a Changelog format

### Removed
- Temporary validation and checkpoint documents
- Version-specific release notes (replaced with CHANGELOG)

## [1.0.0] - 2026-03-08

### Added
- Initial release of Home Assistant Development Power
- Complete toolkit for Home Assistant development with 17 MCP tools
- Auto-configuration of ha-dev-tools-mcp server via uvx
- Comprehensive steering guides for common workflows:
  - Getting started guide
  - File management patterns
  - Template testing workflows
  - Debugging techniques
  - Local development setup
  - Version management
  - Workflow patterns
- Example code snippets for common operations
- Full documentation in POWER.md

### Features
- File operations (read, write, list, delete, upload, download)
- Template testing and validation
- Entity and state management
- Service call execution
- Log access and monitoring
- System information retrieval
- Metadata operations
